# VYPER — Raport Kualitas Aplikasi

> **Buka file HTML di browser** untuk melihat raport kualitas.
>
> Zero dependency. Offline-ready. Dark mode support.

## Konvensi Penamaan

```
raport_app_YYYY-MM-DD.html
```

| File | Tanggal | Deskripsi |
|------|---------|-----------|
| `raport_app_2026-06-04.html` | 2026-06-04 | Baseline — skor awal 71/100 (B-) |
| `raport_app_2026-07-04.html` | 2026-07-04 | *(target: 75/100)* |
| `raport_app_2026-08-04.html` | 2026-08-04 | *(target: 82/100)* |
| `raport_app_2026-09-04.html` | 2026-09-04 | *(target: 88/100)* |
| `raport_app_TEMPLATE.html` | — | Template untuk assessment baru |

## Cara Pakai

1. **Lihat raport terkini**: Buka `raport_app_2026-06-04.html`
2. **Buat assessment baru**: Copy `raport_app_TEMPLATE.html` → `raport_app_YYYY-MM-DD.html`
3. **Update dashboard**: Edit `raport_app_YYYY-MM-DD.html`, tambah ke riwayat skor
